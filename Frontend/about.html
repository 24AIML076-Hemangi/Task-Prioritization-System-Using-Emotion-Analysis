Task Prioritization System Using Emotion Analysis 🤖

About This App / How It Works

This application helps users organize their daily tasks by combining traditional productivity frameworks with emotional context.

Emotion analysis is optional and used only for assistive suggestions.

How the App Works

  • 📝 Step 1 – Add a Task: Type a task in the input field and press Enter.
  • 📅 Step 2 – Assign a Due Date: Select a deadline using the date picker.
  • ⚙️ Step 3 – Choose a Priority Matrix: Select a prioritization method such as the Eisenhower Matrix (Important vs Urgent).
  • 📂 Step 4 – Organize Tasks: Tasks can be grouped into categories like Work, Study, and Personal.
  • 📆 Step 5 – View Tasks by Time: You can track tasks in different views such as Today, Next 7 Days, and Inbox.
  • 🤖 Step 6 – Smart Prioritization: The system helps users decide which tasks are most important using productivity frameworks.
  • 🚀 Step 7 – Stay Organized: Track progress, manage deadlines, and improve productivity.

Why This App Exists

This project was built to help people manage productivity while considering emotional states and cognitive load. It combines structured prioritization with optional emotion input to support better planning.
