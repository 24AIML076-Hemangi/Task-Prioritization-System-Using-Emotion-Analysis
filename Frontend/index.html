About Our App

Task Prioritize System Using Emotion Analysis

This app helps users manage daily work with task planning, reminders, and optional emotion-aware suggestions. It is designed to keep the user in control while making prioritization easier.

How Our App Works

1
Create your tasks.

Add tasks quickly, set deadlines, and keep everything in one dashboard.

2
Choose the right priority.

Use importance, urgency, and prioritization matrices like Eisenhower to organize work clearly.

3
Use optional emotion analysis.

The app can scan emotion only when the user chooses, then suggest what kind of tasks may be easier to handle next.

4
Get reminders and stay on track.

Email and SMS reminders help users notice pending work before deadlines are missed.

Why It Helps

Many task apps only store lists. This one also helps users decide what to do first when they feel overloaded or unsure.

User First

Emotion input is assistive, not controlling. Manual priority choices still matter most, so the app supports decisions instead of replacing them.

Built For Students

The app fits study, personal, and work tasks in one place, which makes it practical for everyday planning and project tracking.

About Us

Task Prioritization System Using Emotion Analysis is an AIML student project focused on combining productivity methods with thoughtful, optional emotion-aware support.

The goal is simple: help users stay organized, reduce overload, and make better daily decisions with clear task structure and friendly reminders.
