Privacy Policy

Last updated: 2026-03-05

This Privacy Policy describes how Task Prioritization System Using Emotion Analysis ("we", "our", "the app") collects and uses information.

Data We Collect

  • Account data: email and password hash (never plain text).
  • Task data: titles, importance, urgency, due/reminder timestamps.
  • Optional emotion scan data: emotion label and confidence.
  • Optional reminder delivery data: phone number, reminder preferences.

How We Use Data

  • Authenticate users.
  • Store and manage tasks.
  • Send reminders by email/SMS if configured.
  • Provide optional emotion-assisted suggestions.

Camera Use

  • The camera is used only when the user manually triggers emotion scan.
  • Images are processed to detect emotion; no image is stored by the server.

Data Sharing

  • If email/SMS reminders are enabled, data is sent to providers (Gmail API/SMTP/SendGrid/Twilio) only to deliver reminders.
  • We do not sell personal data.

Data Retention

Data is stored until the user deletes their account or tasks.
